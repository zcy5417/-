总结可能弹性模子有以下问题：  
1、可能A会变，B、C之间的差异用它们和A的距离来比较不合适  
2、所谓的距离不一定是自己定义的距离，可能是学习出来的某个距离   
3、这种迭代优化都不一定能保证起到啥作用
